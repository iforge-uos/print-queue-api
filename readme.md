This is a "simple" application that will provide an endpoint for the print queue program to access and modify the print queue.

## Requirements
- PostgreSQL database
- Flask
- Flask-sqlalchemy
- Flask-RESTful
- Flask-Migrate
- JSONify
- python-dotenv
- psycopg2
- toml
- marshmallow
- marshmallow-enum
- pycodestyle
- autopep8
